# Market Dashboard Analytics Engine

Historical weekly market analytics platform.

## Features
- 52-week breakout detection
- Sector-wise strength analysis
- Relative Strength Matrix
- Market breadth & participation
- Historical (non-live) dashboard snapshots

## Tech Stack
- Java 21
- Spring Boot
- JPA (Projection-based queries)
- Lombok
- PostgreSQL / MySQL

## Notes
- Minimum timeframe: 1 week
- Dashboard computed from historical data only
